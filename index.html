Validación de Certificados

Sistema institucional de consulta para verificar la autenticidad de certificados emitidos por Lex Justa - Asociación Jurídica.

Consulta de autenticidad

Ingrese el código que figura en el certificado para verificar su autenticidad.

Indicaciones

  1. Ingrese el código completo del certificado.
  2. Presione el botón Validar.
  3. Si el certificado se encuentra registrado, se mostrarán los datos del titular.
  4. Si el código no corresponde al registro, el sistema mostrará el resultado como inválido.
